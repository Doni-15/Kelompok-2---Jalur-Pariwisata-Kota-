Langkah menjalankan program ini:

--- Langkah Awal (Pertama kali buka folder ini) ---
1. Install extensions CMake Tools dari Microsoft
2. Di VS Code, tekan Ctrl+Shift+P untuk membuka Command Palette
3. Ketik: CMake: Select a Kit (atau pilih dari daftar)
4. Cari daftar yang memiliki nama GCC 10.3.0
5. Sekarang, coba klik tombol [Build] di status bar bawah.

--- Langkah setelah melakukan langkah di atas ---
1. Lihat ke Status Bar di bagian paling bawah VS Code Anda
2. Cari tombol yang bertuliskan [Build]
3. Klik tombol [BUILD] itu
4. Cari tombol yang ada icon ▶
5. Klik tombol ▶

Proyek selesai dijalankan.
